# File Encryption and Decryption Tool

A simple Python project that encrypts and decrypts files securely.

## Features
- Encrypt files
- Decrypt files
- Easy to use
- Secure file handling

## Technologies Used
- Python

## How to Run
1. Download the project files.
2. Run the Python script.
3. Choose Encrypt or Decrypt option.
4. Select the file and process it.
